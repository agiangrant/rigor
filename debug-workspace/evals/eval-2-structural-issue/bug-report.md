# Bug Report

**Title**: Users receiving SMS notifications even after disabling them in settings

**Steps to reproduce**:
1. User disables SMS in notification preferences (sets `sms: false`)
2. A security alert is triggered for the user's account
3. User still receives an SMS

**Customer feedback**: "I turned off text messages in my settings but I keep getting them. This is really annoying — I'm on a limited SMS plan."

**Additional context**: Product team is about to add 3 more notification types next sprint (delivery updates, price drop alerts, review requests). Engineering lead mentioned wanting to add WhatsApp and Slack as notification channels in Q3.
